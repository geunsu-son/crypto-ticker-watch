# Privacy Policy for Crypto Ticker Watch

Crypto Ticker Watch is designed to monitor selected cryptocurrency ticker prices from OKX and Binance.

## Data stored locally

The extension stores the following data locally in Chrome using `chrome.storage.local`:

- Selected ticker list
- Selected badge ticker
- Price update interval
- Change calculation basis and timeframe
- Price alert rules, including target price, direction, repeat setting, and triggered status
- Cached ticker list results
- Cached recent price history used to calculate short timeframe changes

## Data sent to third parties

The extension requests public market data from:

- OKX
- Binance Spot API
- Binance Futures API

These requests are used only to search instruments and fetch market prices. Chrome notification permission is used only to show locally configured price target alerts.

## Personal data

The extension does not collect personal information, account information, browsing history, wallet addresses, or trading data.

## Data sharing

The developer does not sell, share, or transfer user data.

## Contact

For questions, contact the developer through the Chrome Web Store listing.
